# Microservice Sample User Service
User service for microservice sample application.

## Setup
You can install dependencies with `yarn`.

```sh
yarn install
```

## Run dev server
Set environments for development server.

```sh
yarn dev
```

## Lint files
Run lint with Eslint.

```sh
yarn lint
```

## Run test
Run test with ava.

```sh
yarn test
```
